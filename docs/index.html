The Coven Modpack

How to Install (using MultiMC)

Download and install MultiMC

Open the settings window

Select the accounts tab

Click "Add Microsoft" (Mojang accounts don't work anymore)

Follow the instructions to log into your Microsoft account

Close the settings window

Click "Add Instance"

Select the "Modrinth" tab

Enter "The Coven" in the search bar and click Search

Select "The Coven Modpack" and press OK

Wait for the application to download the mods

Click "Launch" to test your installation

All done - the modpack will update automatically if anything changes
